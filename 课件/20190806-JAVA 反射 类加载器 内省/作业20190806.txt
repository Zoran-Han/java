1、实现 一个方法：

     public static Object execute(String className, String methodName, Object args[])；

     实现“通过类的名字、方法名字、方法参数调调用方法，返回值为该方法的返回值” 的功能。

     提示：

         方法中的逻辑自行定义。     


2、定义一个接口，方法自定义（至少一个）；定义多个类实现接口（至少两个）；根据要求完成下列功能：[可选]

     要求：创建属性列表.properties文件,文件中配置接口与接口实现类的关系

           格式如下：

               interface.impls = com.***.***.impl.***,com.***.***.impl.***,...

           定义工厂类，构建工厂对象时，创建所有的接口实现类对象；

           工厂类定义方法，可以根据类名获取已经创建的实现类对象，若未找到对象就新建对象；

           定义测试类，定义方法，方法中通过接口访问不同实现类对象。
